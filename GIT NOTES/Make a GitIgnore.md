# MAKE A GITIGNORE

We create a file to ignore.

```
$ cat >> .gitignore
Ignore.txt
CTRL+C
```


We check now.

```
$ cat .gitignore
gitignore.txt
```


We add and commit.


```
$ git add .
$ git commit -m "Git Ignore added"
```

We push to Repository.

```
$ git push -u repositoryname master
```


**GitIgnore process has been completed.**
